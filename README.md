# Enerji Akademisi (Stantec Portal)

Enerji farkındalığını artırmayı amaçlayan interaktif bir eğitim ve kişisel değerlendirme platformu.

## Özellikler

*   **Kişisel Değerlendirme:** 30 soruluk bilgi ve davranış testi.
*   **Detaylı Raporlama:** Kullanıcı seviyesine göre (Düzey 1-3) detaylı analiz ve grafikler.
*   **Kişiselleştirilmiş Eğitim:** Sonuçlara göre önerilen eğitim modülleri.
*   **Modern Arayüz:** Kullanıcı dostu, mobil uyumlu ve şık tasarım.
*   **Gerçek Veriler:** Türkiye geneli enerji verileriyle (Karbon Ayak İzi, Ulusal Hedefler) entegre dashboard.

## Kurulum ve Çalıştırma

Bu proje statik HTML/CSS/JS dosyalarından oluşur.

1.  Dosyaları indirin.
2.  Proje dizininde bir terminal açın.
3.  Bir yerel sunucu başlatın (örn: `npx serve` veya Python ile `python3 -m http.server`).
4.  Tarayıcınızda sunucu adresini (genellikle `http://localhost:3000` veya `http://localhost:8000`) açın.
    > **Not:** JavaScript modülleri (import/export) güvenlik nedeniyle doğrudan dosya sistemi (`file://`) üzerinden çalışmaz.

## GitHub Pages ile Yayına Alma

Bu proje GitHub Pages uyumludur. Repositoriyi GitHub'a yükledikten sonra:
1.  **Settings** > **Pages** sekmesine gidin.
2.  **Branch** olarak `main` veya `master` seçin.
3.  Kaydedin. Sayfanız `https://kullaniciadi.github.io/repo-adi` adresinde yayınlanacaktır.
